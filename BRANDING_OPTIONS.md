# Branding Options for Telemetry Platform

**Created**: 2025-11-20
**Purpose**: Explore naming options for the telemetry logger + lap viewer platform

## Project Vision Summary

- **Primary Use**: Training tool for sim racers to find weak spots and go faster
- **Key Features**:
  - Telemetry capture during races
  - Lap comparison (you vs others)
  - Reference lap library (pre-loaded fast laps by car/track combo)
  - Social sharing potential (viral lap comparisons)
- **Future**: Multi-sim support, potential SaaS platform
- **Audience**: Sim racers focused on self-improvement, not hardcore engineers

## Domain Availability Notes

Before finalizing, check these TLDs in order of preference:
1. `.com` - Best for SaaS/commercial
2. `.gg` - Gaming/esports standard (Guernsey domain)
3. `.racing` - Niche but on-brand
4. `.io` - Tech startup vibe
